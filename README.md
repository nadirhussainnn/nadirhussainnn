      <b>Thesis:</b> Full-Stack Observability in Kubernetes: A Comparative Experimental Study of Dynatrace and
Open-Source Alternatives for Cloud-Native Microservices
      <br/><br/>
      Modern cloud-native applications on Kubernetes generate vast telemetry — metrics, logs, traces, and events — across dozens of loosely-coupled microservices. This thesis empirically evaluates Dynatrace against the open-source stack (Prometheus · Grafana · Loki · Jaeger) on the same real-world application (Bank of Anthos) running on GKE.
      <br/><br/>
      Dynatrace claims full-stack observability with zero instrumentation and <b>AI-powered autonomous root cause analysis (Davis AI)</b> — including anomaly detection, learned baselining, and automatic problem correlation across the full stack. The research evaluates where these claims hold, where gaps exist, and what architectural tradeoffs engineering teams must understand before choosing between a commercial AI-driven platform and a manually-configured open-source alternative.
